# ABook — Предложения по развитию

Файл содержит идеи по улучшению, доработке и расширению приложения. Разбит по категориям, внутри — по приоритету.

---

## 1. ГОЛОС И TTS (первостепенная задача)

### 1.1 Расширение настроек голоса

- [ ] **Скорость по расписанию** — автоматическое замедление речи ночью (например, 0.8x после 22:00) и ускорение днём
- [ ] **Пресеты скорости "по ситуации"** — "Ходьба" (1.2x), "Засыпание" (0.8x), "Фоновое прослушивание" (1.5x), "Изучение" (0.7x)
- [ ] **Тонкая настройка пауз** — отдельные слайдеры для паузы после запятой, точки, конца абзаца, конца главы
- [ ] **SSML-теги для эмоций** — если движок поддерживает, использовать `<emphasis>` и `<prosody>` для выделения диалогов
- [ ] **Авто-пауза на диалогах** — детектировать прямую речь (кавычки, тире) и добавлять микропаузу перед/после
- [ ] **Голосовые шаблоны для персонажей** — разные pitch/rate для реплик разных персонажей (определять по кавычкам/тире + имя)
- [ ] **Нормализация громкости** — автоматическое выравнивание уровня громкости TTS-вывода
- [ ] **Реверберация (EnvironmentalReverb)** — добавить эффект "комната", "зал", "концертный зал" для объёмного звучания
- [ ] **Запись в аудиофайл** — синтезировать книгу/главу в MP3/OGG для оффлайн-прослушивания без TTS (через `synthesizeToFile()`)
- [ ] **Поддержка нескольких TTS-движков** — позволить переключаться между Google TTS, Samsung TTS, RHVoice и др.
- [ ] **A/B тестирование голосов** — воспроизвести один и тот же текст двумя разными голосами для сравнения
- [ ] **Автоматический выбор лучшего голоса** — по языку книги выбирать голос с наивысшим quality и наименьшим latency

### 1.2 Эквалайзер и эффекты

- [ ] **Параметрический эквалайзер** — добавить Q-фактор (ширину полосы) помимо частоты и уровня
- [ ] **Пользовательские пресеты эквалайзера** — сохранение/загрузка пресетов помимо стандартных
- [ ] **Визуализация частотного спектра** — анимированная гистограмма EQ в реальном времени
- [ ] **Компрессор динамического диапазона** — выравнивание тихих и громких фрагментов
- [ ] **Denoise** — шумоподавление для TTS-артефактов
- [ ] **Эффект "наушники/динамик"** — оптимизация звука под тип вывода

### 1.3 Профили голоса

- [ ] **Профиль привязанный к книге** — автоматически загружать определённый профиль при открытии конкретной книги
- [ ] **Профиль привязанный к жанру** — художественная литература → один профиль, техническая → другой
- [ ] **Экспорт/импорт профилей** — делиться профилями через файл (JSON)
- [ ] **Профиль по Bluetooth-устройству** — разные настройки для наушников, колонки, автомобиля
- [ ] **"Голос дня"** — случайный подбор приятных параметров из сохранённых профилей

## 2. ПАРСИНГ И ФОРМАТЫ КНИГ

### 2.1 Улучшение существующих парсеров

- [ ] **Полноценный FB2-парсер** — сейчас читает как plain text. Нужно: парсинг XML структуры `<body>/<section>/<title>/<p>`, определение автора и названия из `<description>`, обложка из `<binary>`, сноски, эпиграфы
- [ ] **Полноценный EPUB-парсер** — через epublib-core или ручной ZipInputStream + Jsoup. Извлечение TOC из NCX/navigation document, обложка, метаданные
- [ ] **Полноценный PDF-парсер** — через iTextPDF. Определение глав по шрифтам/TOC. Обработка двухколоночного текста
- [ ] **Кодировка Windows-1251** — автоматическая детекция для FB2 файлов (из XML declaration)
- [ ] **DJVU-формат** — добавить через OCR или встроенный текстовый слой
- [ ] **MOBI/AZW3** — форматы Kindle
- [ ] **DOC/DOCX** — Word-документы через Apache POI или аналог
- [ ] **RTF** — Rich Text Format
- [ ] **HTML** — парсинг веб-страниц как книг

### 2.2 Интеллектуальное разбиение на главы

- [ ] **Регулярные выражения для глав** — расширенный набор: "Часть", "Part", "Раздел", "Section", римские цифры, нумерация без слова "Глава"
- [ ] **ML-детекция глав** — использовать heuristik по длине пустых строк, заглавных букв, нумерации
- [ ] **Ручная разбивка** — позволить пользователю отметить начало глав вручную
- [ ] **Распознавание оглавления** — если в книге есть TOC-страница (текстовая), парсить её для навигации

### 2.3 Обработка текста перед озвучкой

- [ ] **Замена аббревиатур** — "т.д." → "так далее", "и т.п." → "и тому подобное"
- [ ] **Замена специальных символов** — "—" → пауза, "..." → длинная пауза
- [ ] **Пропуск колонтитулов и номеров страниц** — для PDF
- [ ] **Пропуск сносок** — или озвучка сносок отдельно в конце главы
- [ ] **Словарь произношений** — пользовательские замены: "ОС" → "операционная система", имена собственные
- [ ] **Фильтрация мусора** — удаление повторяющихся пробелов, битых символов, мусорных строк из PDF
- [ ] **Детекция языка по главам** — если книга мультиязычная, переключать TTS-голос

## 3. ТАЙМЕР ЗАСЫПАНИЯ

- [ ] **"Умное засыпание"** — детекция отсутствия взаимодействия (не трогает телефон) + постепенное замедление и затухание
- [ ] **Таймер по главам** — "остановить после 3 глав"
- [ ] **Расписание на каждый день недели** — разные времена для будней/выходных
- [ ] **Будильник-пробуждение** — обратная функция: начать читать в заданное время утром
- [ ] **Интеграция с Do Not Disturb** — автоматически включать DND при активации таймера сна
- [ ] **Fade-in при пробуждении** — плавное нарастание громкости при утреннем включении
- [ ] **Настраиваемая длительность fadeout** — сейчас фиксированные 120 секунд, сделать слайдер 30с–5мин
- [ ] **Вибрация перед засыпанием** — лёгкая вибрация за 1 минуту до остановки как уведомление
- [ ] **Таймер с подтверждением** — "Вы ещё слушаете?" каждые N минут, если нет ответа → пауза
- [ ] **История таймера** — статистика: во сколько обычно засыпает пользователь

## 4. НАВИГАЦИЯ И ПРОГРЕСС

### 4.1 Улучшение навигации

- [ ] **Перемотка по времени** — оценка длительности озвучки главы/книги на основе WPM (слов в минуту) и текущей скорости. Кнопки ±15с, ±30с, ±1м, ±5м в пересчёте на символы
- [ ] **Перетаскиваемый seek bar** — вместо LinearProgressIndicator использовать Slider для прямой навигации
- [ ] **Мини-карта текста** — визуализация всей книги как полоски, текущая позиция подсвечена
- [ ] **Поиск по тексту** — найти фрагмент и перейти к нему
- [ ] **Закладки** — ставить маркеры в произвольных местах, возвращаться к ним
- [ ] **"Вернуться на 1 предложение"** — для случаев когда отвлёкся
- [ ] **Жесты навигации** — свайп влево/вправо для переключения глав, вверх/вниз для громкости
- [ ] **Двойное нажатие на экран** — ±30 секунд (как в видео-плеерах)
- [ ] **Голосовое управление** — "Следующая глава", "Пауза", "Перемотай на 5 минут вперёд"

### 4.2 Прогресс и статистика

- [ ] **Оценка оставшегося времени** — "Осталось ~2ч 15мин" на основе средней скорости озвучки
- [ ] **Статистика прослушивания** — сколько часов/дней слушал, сколько книг прочитано, скорость чтения
- [ ] **Ежедневная цель** — "30 минут в день", прогресс-кольцо
- [ ] **Стрики (серии)** — "Вы слушаете 7 дней подряд!"
- [ ] **Виджет на рабочий стол** — прогресс текущей книги + быстрый play/pause
- [ ] **Уведомление-напоминание** — "Вы давно не открывали книгу X" через N дней
