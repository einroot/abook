# Политика конфиденциальности ABook

**Последнее обновление:** 2026-04-07

## Общие положения

ABook — это Android-приложение для озвучки электронных книг через встроенный синтезатор речи Google TTS. Приложение работает полностью локально на устройстве пользователя.

## Какие данные мы собираем

**Никакие.** Приложение ABook не собирает, не хранит на серверах и не передаёт третьим лицам никаких персональных данных.

## Какие данные хранятся локально на устройстве

- Импортированные вами книги (тексты, метаданные, обложки)
- Позиции чтения в каждой книге
- Настройки голоса и профили
- Расписание таймера сна
- Статистика прослушивания

Все эти данные хранятся исключительно в локальной базе данных устройства и недоступны никому, кроме вас.

## Разрешения приложения

- **Foreground Service / Media Playback** — для фонового воспроизведения
- **Notifications** — для отображения уведомления плеера
- **Schedule Exact Alarm** — для таймера сна
- **Wake Lock** — чтобы телефон не засыпал во время воспроизведения
- **Vibrate** — для вибро-предупреждения таймера сна
- **Access Notification Policy** — для режима "Не беспокоить" при срабатывании таймера

## Internet

Приложение не использует интернет для своей основной функциональности. Исключение: если вы выбираете онлайн-голос Google TTS, то синтез речи может выполняться через серверы Google согласно [политике конфиденциальности Google](https://policies.google.com/privacy).
